## Install dependencies

```bash
pnpm install
pnpm install -g pino-pretty
```

## Run the app

```bash
pnpm run dev
```

```bash
open http://localhost:3000
```

## Migration

### Generate

```bash
pnpm run db:generate
```

### Migrate

```bash
pnpm run db:migrate
```

### Drop

```bash
pnpm run db:drop
```
